## Setting up Development Workflow

To start Sendsile Vendor locally, follow these steps:

1. **Clone the repository:**

    ```
    git clone https://github.com/D-Accubin/sendsile-vendors-fe.git
    ```

2. **Navigate to the project directory:**

    ```
    cd sendsile-vendors-fe
    ```

3. **Install the dependencies:**

    ```
    yarn install
    ```

4. **Run the development server:**

    ```
    npm run dev
    ```

5. **Open your browser and go to http://localhost:5173 to access the platform.**

## Contributing

We welcome to contribute to Sendsile, please follow these steps:

1. Fork the repository.

2. Create a new branch for your feature:

    ```bash
    git checkout -b feature/my-feature
    ```

3. Make your changes and commit them:

    ```bash
    git commit -m "Add my feature"
    ```

4. Push your changes to your fork:

    ```bash
    git push origin feature/my-feature
    ```

5. Open a pull request on the main repository.
